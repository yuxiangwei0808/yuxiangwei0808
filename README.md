# Yuxiang Wei Personal Website

Static GitHub Pages site for Yuxiang Wei.

## Local Preview

Open `index.html` directly in a browser, or run a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

This repository can be published with GitHub Pages from the `main` branch root.

For a root personal website at `https://yuxiangwei0808.github.io`, rename or create the repository as:

```text
yuxiangwei0808.github.io
```

If this repository stays named `yuxiangwei0808`, GitHub Pages will normally publish it as a project site:

```text
https://yuxiangwei0808.github.io/yuxiangwei0808/
```
